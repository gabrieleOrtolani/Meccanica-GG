Dentro la cartella abs trovi i 5 test che con il file REALTIME_abs.m puoi analizzare premendo RUN


Dentro la cartella stop trovi i 5 test che con il file REALTIME_stop.m puoi analizzare premendo RUN



L'obiettivo è fare un analisi di uno dei 5 test, sia per abs e sia per stop (se li analizzi prima ad occhio -> guardi e scegli il grafico più pulito, ricordati che abs ci devono essere meno vibrazioni sull'asseZ)

E poi un analisi che confronta le medie dei test cosi che possiamo scrivere qualcosa di carino nelle conclusioni :)


Il file complete.m è fatto per analizzare in parallelo i due casi, abs e stop, ovvero prende i dati dai due file e li confronta, prendi spunto :)
